# Student Registration System

## Description

A Java-based Student Registration System developed as part of the Syntecxhub Java Programming Internship.

The project allows users to:

* Add Student Records
* View Student Records
* Update Student Information
* Delete Student Records

## Technologies Used

* Java
* ArrayList
* Object-Oriented Programming (OOP)

## Features

1. Add Student
2. View Students
3. Update Student
4. Delete Student
5. Exit
